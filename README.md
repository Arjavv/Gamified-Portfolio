# Gamified Portfolio 🐉

A highly interactive, RPG-themed personal portfolio built with React and Tailwind CSS. 

Level up your experience as you navigate through skills, projects, and quests in a retro-gaming inspired interface!

## 🌐 Live Demo
**Play the live version here:** [https://arjavdariya.vercel.app/](https://arjavdariya.vercel.app/)

## ✨ Features
- **RPG Aesthetic:** Dark, immersive, retro-gaming inspired user interface.
- **Interactive Quests:** Navigate through your "Current State" and "Active Quests" (Projects/Skills).
- **Live Terminal Clock:** A real-time running clock in the terminal header.
- **Fully Responsive:** Adapts seamlessly to mobile phones, tablets, and desktop screens for a perfect viewing experience anywhere.
- **Single Page Application:** Fast routing and seamless page transitions powered by React Router.

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router (BrowserRouter)
- **Icons:** Lucide React & React Icons
- **Deployment:** Vercel

## 🚀 Local Deployment
To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Arjavv/Gamified-Portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Gamified-Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173/`
